## Installation (1-clic OAuth)

Depuis **Intégrations → Shopify → Connecter**, vous êtes redirigé vers Shopify pour valider les scopes. Après approbation, retour automatique dans Actero.

## Scopes demandés

<AccordionGroup>
  <Accordion title="read_orders, read_customers, read_products">
    Lecture seule. Actero consulte les commandes pour répondre au "où est ma commande", les clients pour personnaliser les réponses, les produits pour répondre aux questions techniques.
  </Accordion>
  <Accordion title="read_abandoned_checkouts">
    Pour le playbook `abandoned_cart`. Actero détecte les paniers abandonnés et déclenche les relances.
  </Accordion>
  <Accordion title="write_draft_orders (optionnel, Pro)">
    Permet à l'agent vocal de créer un devis pendant un appel et de l'envoyer par email au client.
  </Accordion>
</AccordionGroup>

Aucun scope de paiement, de modification de commande ou de remboursement n'est demandé. **Actero ne peut pas agir sur votre caisse**.

## Webhooks configurés automatiquement

- `orders/create` → indexation commande pour réponses WISMO
- `orders/updated` → mise à jour du statut tracking
- `carts/abandoned` → déclenche playbook `abandoned_cart`
- `customers/create` → enrichit la base client

Tous signés HMAC SHA-256 avec `SHOPIFY_CLIENT_SECRET`. Vérification constant-time.

## Multi-boutique

Sur le plan Entreprise, une même organisation Actero peut connecter plusieurs boutiques Shopify et les gérer depuis un dashboard unifié. En plan Pro, vous pouvez avoir **un seul domaine Shopify par compte** — créez un second compte Actero si vous gérez plusieurs marques.

## Déconnexion

Intégrations → Shopify → **Déconnecter**. Actero révoque son token d'accès, supprime les webhooks, et arrête immédiatement de lire vos données. Les conversations passées restent dans l'historique (RGPD).

## Problèmes fréquents

<AccordionGroup>
  <Accordion title="L'agent ne trouve pas ma commande #1234">
    Vérifiez que Shopify est en statut **Connecté** et que le client a bien utilisé un email client Shopify valide. Actero matche d'abord par email, puis par numéro de commande.
  </Accordion>
  <Accordion title="Relances panier ne partent pas">
    Vérifiez que `abandoned_cart` est actif dans **Automatisation**, et que votre SMTP ou Gmail est configuré pour l'envoi d'emails sortants.
  </Accordion>
  <Accordion title="Comment tester sans impacter mes vrais clients ?">
    Utilisez **Tester mon agent** (Simulator) dans le dashboard. Les messages simulés n'envoient rien aux vrais clients.
  </Accordion>
</AccordionGroup>
